# QtCreator Theme for NeoVim

<div align="center">
<img src="./.github/pics/screenshot.png" alt="screenshot" />
</div>

An attempt to port theme called Dark (2016) from QtCreator to NeoVim.
Supports Treesitter.

```
:colorscheme qtcreator
```

## Lazy
``` lua
return {
    "Nyvyme/qtcreator.nvim",
    priority = 1000,
}
```
